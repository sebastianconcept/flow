Example 4 - Lists of controllers

This is the Example4Controller.

The Example4Controller is a BindingController. We need it binding so it can show details of the items in the displayed list.

It also has a (sub)controller which is a ListController. These have ListModel as their model. The ListModel has a collection of Thing(s) and the ListController knows how to create instances to present those Thing(s). They get displayed, one per row, by rendering their own ThingRowController instances that are created and removed dynamically when needed.

Thing is class of Mapless that we are just using as example here to create a bunch of models so our ListModel is not empty and you can play with it.
